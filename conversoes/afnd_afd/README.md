# Conversão ANFD para AFD

Para a conversão entre um AFND para um AFD a técnica de "agrupamento" dos estados pode ser utilizados. A tabela `tabela_de_conversao.ods` contém um exemplo de conversão.

> O exemplo converte uma Λ-transição para o AFD, porém, isto foi inserido apenas como forma de exemplo, onde o pensamento pode ser aplicado para todo e qualquer caso.
